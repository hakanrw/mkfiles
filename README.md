# mkfiles

makefiles for small C and C++ projects.

## contents

`c/Makefile.base`: base makefile for C projects

`cpp/Makefile.base`: base makefile for C++ projects

`example/`: examples projects using these makefiles

## usage

create new makefile with contents

```
TARGET_NAME=<some_target_name>

include <base_makefile_location>
```
done.
